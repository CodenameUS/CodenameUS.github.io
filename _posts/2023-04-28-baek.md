요즘 파이썬만 사용하다가 이 문제를 C언어로 풀려고 해보니 꽤(?) 머리를 썼다.

쉬운문제인데 10분정도 머리를 쓴 것같다..

```c
#include <stdio.h>

int main(void)
{
    int N;

    scanf("%d", &N);


    
        for (int i = N; i > 0; i--) {
            for (int j = 1; j < i; j++) {
                printf(" ");
            }
            for (int k = N; k >= i; k--) {
                printf("*");
            }
            printf("\n");
        }

    
}
```
